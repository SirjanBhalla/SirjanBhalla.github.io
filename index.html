Project Details

Real Time Call Center KPI Architecture

View Interactive Dashboard

Context

In high volume collections environments, leadership often struggles to identify patterns across vast datasets. I simulated a realistic business environment to bridge the gap between raw call logs and executive decision support.

Action

  • Data Engineering: Used the Faker library in Python to generate a sophisticated dataset that mirrored industry trends.
  • Business Logic: Programmed specific constraints for holiday based behavior and time of day trends to avoid statistical flatness.
  • Visualization: Designed a Tableau dashboard with drill down capabilities to track RPC rates, payment conversions, and silence time.

Result

The project successfully translated complex operational trends into a standardized reporting framework. This architecture allows managers to monitor agent performance and call outcomes with real time clarity, reducing the manual effort required for data exploration.

Project Details

AI Image Descriptor

View Code on GitHub

Context

Generating accurate text descriptions for images requires a deep understanding of visual patterns. I developed this AI system to automate image captioning with a focus on model performance and security.

Action

  • Model Fine Tuning: Optimized a BLIP architecture using a dataset of 20,000 images to improve description relevance.
  • Performance Evaluation: Implemented a testing framework using BLEU scores to objectively measure the quality of generated text.
  • Secure Deployment: Built an interactive Streamlit application featuring a session based authentication system for secure user access.

Result

Achieved a competitive 0.33 BLEU score, demonstrating the model ability to handle complex outdoor scenes and human behavior. The final product provides a scalable foundation for automated alt text and visual data indexing.

Graduate Admission Predictor

Launch Interactive App View Code on GitHub

Context

Navigating graduate school applications involves significant uncertainty. I built this predictive tool to help applicants quantify their admission chances based on standardized academic metrics and research profiles.

Action

  • Neural Network Architecture: Engineered a Keras regression model to process multi dimensional inputs including GRE, TOEFL, and CGPA scores.
  • Streamlit Integration: Developed a web application that translates complex probability scores into intuitive categories like High Chance or Low Chance.
  • Reproducible Research: Implemented a robust data pipeline and random seeding to ensure consistent, reliable predictions with an R squared score of 0.74.

Result

The application successfully bridges the gap between raw statistical data and user friendly guidance. It provides prospective students with an immediate, data driven benchmark for their UCLA graduate program applications.

Project Details

Real Estate Price Predictor

Launch Interactive App View Code on GitHub

Context

Real estate pricing is influenced by a complex mix of property features and market cycles. I built this tool to provide homeowners and investors with an objective, data driven estimate that considers both house specific details and the wider economic climate.

Action

  • Ensemble Learning: Implemented a Random Forest regression model to handle the complex, non linear relationships between property tax, insurance, and sale prices.
  • Feature Engineering: Incorporated economic indicators such as "Recession" and "Popularity" to refine predictions beyond standard physical dimensions like bedrooms and lot size.
  • Interactive UI: Developed a Streamlit application with intuitive input forms and tooltips to ensure the model remains accessible and transparent for non technical users.

Result

The model achieved a high level of reliability with an R squared score of 0.79, successfully capturing nearly 80 percent of the price variance. This provides a clear, formatted valuation that helps users navigate market fluctuations with greater confidence.
